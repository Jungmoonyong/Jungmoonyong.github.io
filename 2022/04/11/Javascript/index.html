변수

변수는 어떤 정보에 이름을 붙여 저장하고 싶을때 사용합니다
; 은 한줄이 끝났다는 의미로 사용해줍니다

name 이라는 변수는 Mike 라는 값이 들어가 있는데 Mike 는 따옴표로 감싸져 있는것을 확인할 수 있는데 이유는 문자형이기 때문입니다

class = “수업” 이라는 변수를 선언했을때 오류가 날 수 있습니다
왜냐하면 class 는 이미 자바스크립트에서 사용하는 단어 ( 예약어 ) 이기 때문입니다

alert() 는 경고창을 띄우는 함수입니다
console.log() 는 로그를 찍는 함수입니다

혼자 만드는 프로젝트라면 괜찮지만 여러 개발자가 만드는 프로젝트 라면 만약 다른 개발자가 name 이라는 변수를 사용하게 되면 마지막으로 선언된 name 변수로 덮어씌어 지게 됩니다

이때 let , const 를 사용할 수 있습니다

let

다음과 같은 코드를 실행하면 이미 name 변수는 선언되어있다는 오류 구문을 띄우게 되면서 다른 개발자가 이 변수를 사용하고 있다는 것을 알고 다른 이름으로 변수를 선언할 것입니다

최초로 선언하는 모든 변수에 let 을 붙인다면 에러를 통해 그 변수를 이미 사용하고 있는지 알 수 있을 것입니다

const

const 는 절대로 바뀌지 않는 상수를 입력할때 사용합니다
const 로 선언된 변수를 바꾸려고 하면 에러가 발생합니다

그래서 보통 파이 , 최댓값 등 바뀌지 않는 값을 사용할때 사용합니다
다른 개발자들에게 상수란걸 알리기 위해 대문자로 변수를 선언할 수 있습니다

팁은 모든 변수를 const 로 선언하고 나중에 변경될 여지가 있는 변수는 let 으로 바꿔줍니다

  1. 변수는 문자와 숫자 $ 와 _ 만 사용
  2. 첫글자는 숫자가 될 수 없습니다
  3. 예약어는 사용할 수 없습니다
  4. 가급적 상수는 대문자로 알려주세요
  5. 변수명은 읽기 싶고 이해할 수 있게 선언

자료형

문자형

문자형은 세가지 방식으로 작성할 수 있습니다 ( “ , ‘ , 백틱 )
만약 ‘ 만 써서 표현하고 싶으면 \ 를 통해 특수문자로 인식시킬 수 있습니다
` 은 문자열 내부에 변수를 표현해줄때 사용할 수 있습니다

숫자형

숫자형은 연산이 가능합니다
만약 숫자를 0 으로 나누면 무한값을 얻을 수 있습니다
만약 문자를 숫자로 나누면 Not a Number 이라는 문구가 찍히게 되는데 숫자가 아니라는 뜻입니다

Boolean

boolean 는 논리적인 요소를 나타내고 true 는 참 false 는 거짓입니다
name 이 Mike 인지 체크하고 age 가 40 보다 큰지 체크합니다

null 과 undefined

null 은 존재하지 않는 값을 의미합니다
undefinded 는 값이 할당되지 않았다는 것을 의미합니다

변수를 선언만 하고 아무것도 할당하지 않으면 undefined 를 출력합니다
만약 변수에 null 을 할당한다면 user 가 존재하지 않는다면 의미로 이해하면 됩니다

typeof 연산자

typeof 연산자는 변수의 자료형을 알 수 있습니다
null 이 object 라고 나오는데 object 는 객체형을 의미합니다
null 은 객체가 아니고 자바스크립트 초기버전의 오류입니다

대화상자

// 는 주석입니다

alert 알려줌
prompt 입력 받음
confirm 확인 받음

alert() 함수가 실행되면 메세지를 띄우게 됩니다
prompt() 함수는 사용자에게 어떤 값을 입력 받습니다
confirm() 함수는 확인받을때 사용할 수 있습니다

prompt() 함수는 디폴트 값도 입력할 수 있습니다
함수를 실행하는 괄호 안에서 들어가는 값들을 인수라고합니다

  1. 스크립트 일시 정지
  2. 스타일링 X

형변환

String() 은 문자형으로 변환하고 앞에는 대문자로 써줘야합니다
Number() 은 숫자형으로 변환하고 앞에는 대문자로 써줘야합니다
Boolean() 은 불린형으로 변환

괄호를 써준 이유는 사칙연산으로 인해 나누기 먼저 진행되기 때문입니다
prompt 로 받은값은 문자형으로 받기 때문에 90 과 80 이 더해지면
9080 이 되게 되고 숫자형이 아니더라도 나누기는 숫자형으로 자동 변환되기 때문에 4540 이 나오게 됩니다

false 는 숫자 0 , 빈 문자열 “ , null , undefined , NaN 은 모두 false 가 됩니다

연산자

나머지는 2 로 나눈 나머지가 1 인지 0 인지에 따라 홀수 짝수로 나눌 수 있고
5 보다 작은수를 얻고 싶으면 아무리 큰 수라도 5 로 나눈 나머지를 구해보면 0 ~ 4 까지 값만 얻을 수 있습니다

거듭제곱은 ** 를 사용해 줄 수 있습니다

비교 연산자 , 조건문

== 는 동등 연산자입니다
=== 는 일치 연산자 이고 타입까지 비교합니다

조건문은 어떤 조건에 따라 이 후 행동이 달라지게 하는 행동을 합니다

논리 연산자

OR 는 여러개 중 하나라도 ture 면 true
AND 는 모든값이 true 면 true
Not 은 ture 와 false 를 반대값으로 바꿔줌

반복문

반복문은 동일한 작업을 여러번 반복할 수 있습니다
for 문은 ; 로 구분하며 초기값 , 조건 , 코드 실행 후 작업 순서로 실행됩니다

do while 은 조건문을 아래로 옮길 수 있습니다

break 는 코드를 멈추고 반복문을 빠져나옵니다
continue 는 코드는 멈추지만 다음 반복으로 점프합니다

switch

switch 는 case 가 다양할 경우 간결하게 쓸 수 있습니다
