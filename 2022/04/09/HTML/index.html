HTML Syntax

Jungmoonyong
2022-04-09
Language
Language

Title

title 태그는 웹 페이지의 제목을 나타낼 수 있습니다

Meta

meta 태그는 웹 페이지의 보이지 않는 정보를 제공하는데 사용할 수 있습니다

Br

br 태그는 줄바꿈을 할 수 있습니다

P

p 태그는 하나의 문단을 만들 수 있습니다

B

b 태그는 글자를 굵게 표시할 수 있습니다
strong 태그도 같은 기능을 합니다

I

i 태그는 글자를 기울여서 표시하는 태그입니다

H#

h1 , h2 , h3 , h4 , h5 , h6 태그를 묶어서 h# 태그라고 작성하였습니다
h# 태그는 섹션 , 문장의 제목을 나타내며 숫자가 작을 수록 글자의 크키가 커집니다

A

a 태그는 하이퍼 링크를 걸어주는 태그입니다

href : 클릭시 이동 할 링크
target : 링크를 여는 방법
_self : 현재 페이지
_blank : 새 탭
_parent : 부모 페이지로 iframe 등이 사용된 환경에서 쓰입니다
_top : 최상위 페이지로 iframe 등이 사용된 환경에서 쓰입니다
프레임이름 : 직접 프레임 이름을 명시해서 사용할 수도 있습니다

Img

img 태그는 이미지를 삽입할 수 있습니다

src : 이미지의 경로
width : 이미지 가로 크기
height : 이미지 세로 크기

Table

table 태그는 표를 만드는 태그입니다

tr : 표의 행을 나타냅니다
td : 표의 열을 나타내며 tr 태그 하위에 위치합니다

일반적인 표는 제목줄을 가지고 있는 경우가 많습니다
table 태그도 제목줄을 지원하며 th 태그를 사용할 수 있습니다

Div

div 태그는 레이아웃을 나누는데 주로 쓰입니다

Span

span 태그는 CSS 와 함께 사용합니다

Li

li 태그는 목록을 만들 수 있습니다
이 태그는 단독으로 쓰이지 않으며 ul , ol 태그 내부에 들어갑니다

ol : 번호를 메겨 순서가 있는 목록을 만듭니다
ul : 순서없이 모양으로 목록을 만듭니다

Form

form 태그는 웹 페이지에서의 입력 양식을 의미합니다

텍스트 필드에 글자를 입력하거나 체크박스나 라디오 버튼을 클릭하고 제출 버튼을 누르면
백엔드 서버에 양식이 전달되어 정보를 처리하게 됩니다

name : 폼의 이름
action : 폼 데이터가 전송되는 백엔드 url
method : 폼 전송 방식 GET / POST

Input

input 태그는 전체 양식을 의미하여 화면에 보이지 않는 추상적인 태그입니다
type 속성을 통해 종류를 나타내며 name 을 통해 데이터의 이름 value 를 통해 기본 값을 지정합니다

text : 일반 문자
password : 비밀번호
button : 버튼
submit : 양식 제출용 버튼
reset : 양식 초기화용 버튼
radio : 한개만 선택할 수 있는 컴포넌트
checkbox : 다수를 선택할 수 있는 컴포넌트
file : 파일 업로드
hidden : 사용자에게 보이지 않는 숨은 요소

Select

select 및 option 은 드롭다운 리스트를 만드는 태그입니다
